Water Level Monitoring System

An IoT solution for real-time monitoring of tank and reservoir water levels using ultrasonic sensors and GSM alerts.

Overview

The Water Level Automation System helps manage water in overhead and underground tanks efficiently through:

  • Ultrasonic Sensor for Underground Tank
  • Float Sensors for Overhead Tank
  • Automatic Motor Activation based on water levels
  • Buzzer Alerts for Full/Empty tanks
  • Manual and Auto Modes via App Control
  • Real-time water level display on LEDs and App

Technical Documentation

Architecture:
  • Ultrasonic Sensor HC-SR04 in underground tank
  • Float Switch Sensors in overhead tank
  • Motor controlled by relay module and Raspberry Pi
  • Mobile App Interface for mode switching and monitoring
  • Failsafe buzzer for overflow/low level warning
