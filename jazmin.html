Jazmin Security System

A smart IoT-based anti-theft system featuring face recognition, motion sensors, gas & fire detection, and app-based control.

Overview

The Jazmin Security system uses Raspberry Pi 4B and SIM800A GSM Module to monitor home security remotely through an app. It features:

  • Face Recognition with camera
  • Motion Detection Sensors
  • Gas and Fire Detection
  • IVRS Calls and SMS Alerts
  • Emergency Buzzer System
  • Full mobile app control

Technical Documentation

Architecture:
  • Raspberry Pi 4B as main controller
  • Camera module for facial recognition and PIR sensors for motion detection
  • MQ2 Gas Sensor and Flame Sensor for smoke/fire detection
  • SIM800A GSM Module for IVRS calls and SMS alerts
  • App developed with React Native.
