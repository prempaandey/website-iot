Authen - Face Authentication System

Introduction

Authen is a facial authentication project using a camera, LED indicators, and an LCD display. The system captures the photo of the person automatically when they stand in front of the camera and matches it against a stored set of images.

Working Principle

  • Camera captures the image automatically when a person is detected.
  • Facial recognition compares the new image with saved authorized images.
  • If match found:
    • Green LED lights up.
    • LCD shows the candidate's name.
  • If not matched:
    • Red LED lights up.
    • LCD shows "Not Recognized".
